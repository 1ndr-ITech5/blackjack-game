The BlackJack Game

A simple and interactive Blackjack game coded in Python.

The game includes:

  -Betting system
  
  -Player and computer turns
  
  -Automatic Ace (11 → 1) adjustment
  
  -Score calculation
  
  -Win/lose conditions
  
  -Looping rounds with balance updates



Features of the game

✔ Player Options

  -Draw a new card (y)
  
  -Stand/pass (n)
  
  -Cards are randomly selected from a standard deck
  
  -Aces automatically convert from 11 → 1 to avoid going over 21

✔ Computer AI

  -Computer draws cards until its score reaches 17 or more
  
  -Automatically adjusts Aces just like the player

✔ Betting System

  -Player starts with a default balance
  
  -Can place bets each round
  
  -Balance updates after win/lose
  
  -Prevents playing with insufficient funds
